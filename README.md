# 🌡️ South America: Heating Up?
 
An interactive data visualization tracking surface temperature changes across South American countries from **1961 to 2024**.
 
## Live Demo
 
[View the visualization](https://be910.github.io/S.America_Climate_IntPlot)

## Overview
 
This project explores how surface temperatures have shifted across South America over six decades. A year slider lets users step through history and watch the warming and cooling patterns evolve country by country.
 
## Features
 
- **Year slider** — scrub from 1961 to 2024; the chart and summary stats update in real time
- **Live summary cards** — displays average temperature change, highest warming value, and count of warming countries for the selected year
- **Diverging bar chart** — countries colored red (warming) or blue (cooling) relative to baseline, sorted by magnitude with exact °C values labeled on each bar
 
## Data Source

Macroeconomic Climate Indicators Dashboard (IMF)
 
## Tech Stack
 
- **D3.js** — diverging bar chart, scales, and axis
- **HTML/CSS/JS** — interactive slider and summary stat cards
